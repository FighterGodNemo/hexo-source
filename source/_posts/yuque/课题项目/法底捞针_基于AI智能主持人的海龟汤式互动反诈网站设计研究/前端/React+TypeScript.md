## React：
前端 UI 框架，用组件方式搭页面。你把页面拆成很多组件（按钮、卡片、页面），状态变化时 React 自动刷新界面。

## TypeScript (TS)：
JavaScript 的“类型增强版”。能提前发现很多错误（比如传错参数类型），大型项目更稳、更好维护。

## 常见命令：
1. npm start：开发模式启动，热更新，适合写代码。（开发）
2. npm run build：打生产包，输出到 build/，用于部署。（上线）
3. npm test：运行测试。
