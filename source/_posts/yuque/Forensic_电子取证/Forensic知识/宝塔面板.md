### <font style="color:rgb(31, 45, 61);">忘记Linux 3.X/4.x/5.x/6.x/7.x 宝塔面板密码的解决方案</font>
+ <font style="color:rgb(76, 73, 72);">进入ssh，推荐先安装 堡塔SSH客户端 (免费/简单/中文/多屏) ，输入以下命令重置密码（把命令最后面的“testpasswd”替换成你要改的新密码）</font>
    - <font style="color:rgb(76, 73, 72);">注：若是debian/ubuntu用户，请使用有root权限的账户去执行这条命令</font>

| ```plain cd /www/server/panel && btpython tools.py panel testpasswd ```  |
| --- |


+ <font style="color:rgb(76, 73, 72);">如果提示多次登录失败，暂时禁止登录 请输入以下命令 清除登录限制</font>

| ```plain rm -f /www/server/panel/data/*.login ```  |
| --- |


+ <font style="color:rgb(76, 73, 72);">新装面板用户获取默认账号密码命令：</font>

| ```plain bt default ```  |
| --- |


+ <font style="color:rgb(76, 73, 72);">6.x及以后版本可输入</font>`<font style="color:rgb(244, 116, 102);background-color:rgba(27, 31, 35, 0.05);">bt</font>`
