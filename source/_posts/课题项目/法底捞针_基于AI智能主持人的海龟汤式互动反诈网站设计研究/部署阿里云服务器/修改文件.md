1. 改了后端代码（比如 backend/src/app.js）

  cd /var/www/project/backend

  sudo systemctl restart project-backend

  sudo systemctl status project-backend --no-pager

  可再检查：

  curl -s [https://lawturtlesoup.top/api/health](https://lawturtlesoup.top/api/health)

  sudo journalctl -u project-backend -n 100 --no-pager



2. 改了前端源码（frontend/src/*）

  cd /var/www/project/frontend  
  npm run build  
  sudo nginx -t && sudo systemctl reload nginx

  （静态文件是 build/，不重启后端）

3. 改了 Nginx 配置

  sudo nginx -t  
  sudo systemctl reload nginx

4. 改了 systemd 服务文件（*.service）

  sudo systemctl daemon-reload  
  sudo systemctl restart lawturtlesoup-backend

  你这次如果只是改了那段 app.js 日志文案，执行第 1 步就行。
